# 2048 Game - Vanilla JavaScript

A modern, responsive version of the classic 2048 puzzle game built using **Object-Oriented Programming (OOP)** principles in JavaScript.

## 🚀 Live Demo
https://sahilkokane.github.io/2048-Game-JS/

## ✨ Features
- **Core Logic:** Fully functional slide and merge mechanics.
- **High Score Persistence:** Saves your best score using `localStorage`, so it stays even after refreshing the page.
- **Responsive UI:** Clean design with a focus on User Experience (UX).
- **Game Over State:** A stylish frosted-glass (blur) overlay with a final score display.
- **Mobile Friendly:** Built with a layout that works across different screen sizes.

## 🛠️ Tech Stack
- **HTML5:** Structure of the game.
- **CSS3:** Styling, Grid layout, and Blur animations.
- **JavaScript (ES6+):** Logic implementation using Classes (OOP).

## 🎮 How to Play
1. Use your **Arrow Keys** (Up, Down, Left, Right) to move the tiles.
2. When two tiles with the same number touch, they **merge into one!**
3. Add them up to reach **2048** (and beyond)!
4. If the board fills up and no moves are possible, the game is over.

## 📂 Project Structure
- `index.html` - The entry point and UI structure.
- `style.css` - Custom styles and frosted-glass effects.
- `script.js` - Game logic, state management, and event handling.
